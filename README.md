# Selenium Java Automation Project

A Selenium WebDriver automation project built using Java and Maven. This project automates the end-to-end purchase flow on the SauceDemo website, including login, product verification, adding items to the cart, checkout, and order confirmation.

---

## 🚀 Technologies Used

- Java
- Selenium WebDriver
- Maven
- WebDriverManager
- ChromeDriver
- VS Code

## ✅ Test Scenario

The automation performs the following steps:

1. Open the SauceDemo website
2. Login with valid credentials
3. Verify the Products page
4. Add two products to the shopping cart
5. Open the cart
6. Complete the checkout process
7. Verify the order confirmation message

## ⚙️ Prerequisites

Before running this project, install:

- Java JDK 17 or later
- Maven
- Google Chrome
- VS Code or IntelliJ IDEA

---

## 📥 Clone the Repository

```bash
git clone https://github.com/Salmanzzoha/Selenium-Test-2026-1.git
```

Move into the project folder:

```bash
cd Selenium-Test-2026-1
```

---

## 📦 Install Dependencies

```bash
mvn clean install
```

---

## ▶️ Run the Project

```bash
mvn exec:java
```

Or simply run:

```
SauceDemoProject.java
```

from your IDE.
